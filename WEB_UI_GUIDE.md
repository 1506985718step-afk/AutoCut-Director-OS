# Web UI 使用指南

## 🎨 全新的 Web 界面

AutoCut Director 现在提供了一个现代化、用户友好的 Web 界面，让视频剪辑变得更加简单直观！

---

## 🚀 快速开始

### 1. 启动服务

```bash
cd autocut-director
python run_server.py
```

### 2. 访问 Web UI

在浏览器中打开: **http://localhost:8787**

---

## 📱 界面功能

### 🎯 四个主要标签页

#### 1️⃣ 上传视频

**功能**:
- 拖拽或点击上传视频文件
- 自动提取音频用于 ASR 转录
- 创建 Job 目录结构

**使用步骤**:
1. 点击或拖拽视频文件到上传区域
2. 选择是否提取音频（默认勾选）
3. 点击"开始上传"
4. 等待上传完成，获取 Job ID

**支持格式**: MP4, MOV, AVI

---

#### 2️⃣ AI 生成

**功能**:
- 上传 scenes.json 和 transcript.json
- 选择剪辑风格（抖音、B站、YouTube、快手）
- AI 自动生成剪辑脚本
- 下载生成的 DSL 文件

**使用步骤**:
1. 上传 scenes.v1.json 文件
2. 上传 transcript.v1.json 文件
3. 选择剪辑风格或自定义风格描述
4. 点击"AI 生成剪辑脚本"
5. 等待 AI 生成（使用 DeepSeek）
6. 查看生成的 DSL
7. 下载 DSL 文件

**可用风格**:
- 🔥 抖音爆款风格：节奏快、文字多、强调关键词
- 📚 B站知识区风格：节奏适中、保留完整讲解、突出知识点
- 🎥 YouTube Vlog风格：自然流畅、保留情感、适度剪辑
- ⚡ 快手热门风格：接地气、情感强、节奏紧凑

---

#### 3️⃣ 任务状态

**功能**:
- 查询 Job 状态
- 查看任务详情
- 获取执行结果

**使用步骤**:
1. 输入 Job ID
2. 点击"查询状态"
3. 查看任务详情

---

#### 4️⃣ 关于

**功能**:
- 查看系统介绍
- 了解核心功能
- 查看使用流程
- 访问 API 文档

## 📋 完整工作流程

### 方式 1: 使用 Web UI（推荐）

```
1. 上传视频
   ↓
2. 在 Resolve 中进行场景切点检测
   ↓
3. 导出 EDL 和 ASR 转录
   ↓
4. 在 Web UI 中上传 scenes.json 和 transcript.json
   ↓
5. 选择剪辑风格，AI 生成 DSL
   ↓
6. 下载 DSL 文件
   ↓
7. 在 Resolve 中执行剪辑
   ↓
8. 导出成片
```

### 方式 2: 使用 API

访问 API 文档: **http://localhost:8787/docs**

## 📝 常见问题

### Q: 为什么上传很慢？

A: 视频文件较大，上传需要时间。建议使用有线网络。

### Q: AI 生成需要多久？

A: 通常 10-30 秒，取决于视频长度和网络状况。

### Q: 支持哪些浏览器？

A: 推荐使用 Chrome, Edge, Firefox 最新版本。

### Q: 可以在手机上使用吗？

A: 可以，界面是响应式的，但建议在电脑上使用以获得最佳体验。

### Q: 如何查看 API 文档？

A: 访问 http://localhost:8787/docs
